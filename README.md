# ⚡ Logic Gate Challenge (ECE Edition)

An interactive, web-based digital logic quiz game designed with a sleek cyberpunk/neon aesthetic. Test your conceptual understanding of fundamental logic gates through real-time truth tables and schematic symbol identification.

🎯 **Live Demo:** [Launch Simulation](https://anmolsaxena08.github.io/game/)

---

## 🚀 Features

* **7 Core Logic Gates:** Covers `AND`, `OR`, `NOT`, `NAND`, `NOR`, `XOR`, and `XNOR`.
* **Dynamic UI:** Cyberpunk-inspired terminal look featuring glowing vector symbols (SVG) and customized digital styling.
* **Streak & Level System:** Tracks your consecutive correct answers and unlocks multipliers to scale up your final score.
* **Instant Feedback Engine:** Provides a quick breakdown of each gate's Boolean logic behavior immediately after answering.
* **Completely Responsive:** Smoothly styled using CSS Grid and standard Viewports to support desktops, tablets, and smartphones.

---

## 🛠️ Tech Stack

* **Frontend Structure:** HTML5
* **Styling & FX:** CSS3 (Featuring Custom Google Fonts: *Orbitron* & *Share Tech Mono*, custom animations, scanline overlays, and neon glow drop-shadows)
* **Game Logic & SVGs:** Vanilla JavaScript (ES6+)

---

## 🎮 How to Play

1. Observe the generated **Truth Table** on the left panel.
2. Inspect the raw **Gate Schematic Symbol** on the right panel.
3. Select the correct gate name from the 4 dynamic choices provided below.
4. Maintain a streak above 3 to unlock score bonuses. Complete all 10 stages to receive your logic efficiency evaluation.
